# 🎬 演示指南

## 🚀 快速开始

这个项目已经可以直接运行，所有UI和交互都已完成！

### 启动应用

```bash
# 如果还没安装依赖
npm install

# 启动开发服务器
npm run dev
```

应用将在 `http://localhost:5173` 启动（或其他可用端口）。

## 🎯 功能演示流程

### 1. 首页浏览

打开应用后，你会看到：

- ✅ **左侧导航栏**：黑色背景，"Artlist" logo，各种导航项
- ✅ **顶部栏**：搜索框，"Subscribe Now" 黄色按钮
- ✅ **主工作区**：大标题 "Transform your ideas into stunning visuals"
- ✅ **底部作品区**：展示视频缩略图网格

### 2. 上传产品图片

**方式一：拖拽上传**
1. 准备一张产品图片（建议：化妆品、喷雾瓶、护肤品等小物件）
2. 直接拖拽到上传区域的虚线框内
3. 等待上传完成

**方式二：点击上传**
1. 点击上传区域
2. 在文件选择器中选择图片
3. 确认选择

**上传后：**
- 图片会显示在上传区域
- 下方出现三个下拉选择器：Model / Duration / Resolution
- 右侧出现 "Animate for Free" 黄色按钮
- **AI 导演面板自动从右侧滑出！** 🎉

### 3. 与 AI 导演对话

AI 导演面板打开后：

**第一步：产品名称**
- AI 会问："请告诉我这是什么产品？"
- 输入产品名称，例如："清新口气喷雾"
- 按 Enter 或点击发送按钮

**第二步：目标市场**
- AI 会显示可点击的 Chips 按钮：
  - 🇨🇳 中国
  - 🇮🇩 印尼
  - 🇺🇸 美国
  - 🌏 全球
- 点击任一选项

**第三步：创意描述**
- AI 会问更多关于视频创意的问题
- 你可以自由描述想要的场景、情感、动作等
- 例如："办公室场景，吃完午饭后偷偷使用"

**对话特点：**
- 🟡 用户消息：黄色气泡，右对齐
- ⚫ AI 消息：灰色气泡，左对齐
- 💭 思考状态：三个跳动的黄色小点
- 🔘 交互 Chips：可点击的圆角按钮

### 4. 生成视频

对话完成后：

1. **生成视频按钮出现**
   - 位置：AI 导演面板底部
   - 样式：黄色渐变，显示 "生成视频 (50 Credits)"

2. **点击生成**
   - 按钮变为灰色，显示 "生成中..."
   - 有旋转的加载图标
   - AI 发送消息："🎬 正在调用 Sora-2 视频生成模型..."

3. **等待完成**
   - 模拟生成过程约 5 秒
   - 完成后 AI 发送成功消息
   - 显示视频详情（时长、分辨率、风格）

### 5. 查看作品

- 滚动到底部的 "Explore" 或 "My Creations" 标签
- 查看视频网格
- 鼠标悬停在视频卡片上会显示 "View" 按钮

## 📱 多设备测试

### 桌面端（推荐）
- 宽度：≥ 1440px
- 体验最佳

### 平板端
- 宽度：768px - 1439px
- AI 导演面板会覆盖部分主区域

### 移动端
- 宽度：< 768px
- 侧边栏可能需要收起
